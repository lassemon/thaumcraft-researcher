# thaumcraft-researcher

### Installation
1. Install Node and npm (=> 0.10.0, => 2.4.1 respectively)
2. ```$ npm install --global gulp bower```
3. ```$  npm install``` in project root

### Starting app
```
$ npm run start
```

### Runnin tests
```
$ npm run test
```

### Packaging app
```
$ npm run package
```
